# 🚀 Инструкция по добавлению Google Gemini в Cline

## Шаг 1: Получение API ключа Google Gemini

1. Перейдите на сайт [Google AI Studio](https://aistudio.google.com/app/apikey)
2. Войдите в свой Google аккаунт
3. Нажмите **"Get API Key"** или **"Create API Key"**
4. Скопируйте полученный ключ (формат: `AIza...`)

---

## Шаг 2: Настройка Gemini в Cline

### Вариант A: Через настройки VS Code

1. Откройте VS Code
2. Нажмите `Ctrl + Shift + P` (или `Cmd + Shift + P` на Mac)
3. Введите: `Preferences: Open User Settings (JSON)`
4. Добавьте следующую конфигурацию:

```json
{
  "cline.apiProvider": "gemini",
  "cline.geminiApiKey": "ВАШ_API_КЛЮЧ_ЗДЕСЬ",
  "cline.geminiModel": "gemini-2.0-flash-exp"
}
```

### Вариант B: Через интерфейс Cline

1. Откройте панель Cline в VS Code (иконка в боковой панели)
2. Нажмите на иконку настроек ⚙️ (обычно в верхнем правом углу панели Cline)
3. Выберите **API Provider**: `Gemini`
4. Вставьте ваш API ключ в поле **API Key**
5. Выберите модель (рекомендуется `gemini-2.0-flash-exp` или `gemini-1.5-pro`)

---

## Шаг 3: Доступные модели Gemini

| Модель | Описание | Рекомендуется для |
|--------|----------|-------------------|
| `gemini-2.0-flash-exp` | Экспериментальная, быстрая | Общие задачи, быстрые ответы |
| `gemini-1.5-pro` | Продвинутая, большой контекст | Сложные задачи, анализ кода |
| `gemini-1.5-flash` | Быстрая, эффективная | Простые задачи |

---

## Шаг 4: Проверка работы

1. Откройте Cline
2. Напишите тестовый запрос: "Привет! Ты работаешь?"
3. Если Gemini отвечает - всё настроено правильно! ✅

---

## 🔧 Устранение проблем

### Ошибка: "Invalid API Key"
- Проверьте, что ключ скопирован полностью
- Убедитесь, что API включен в Google Cloud Console

### Ошибка: "Rate limit exceeded"
- Gemini имеет бесплатный лимит: 15 запросов в минуту
- Подождите минуту или обновите план

### Модель не отвечает
- Попробуйте другую модель из списка
- Проверьте интернет-соединение

---

## 💡 Полезные советы

1. **Бесплатный план Gemini** включает:
   - 15 запросов в минуту
   - 1500 запросов в день
   - 1 миллион запросов в месяц

2. **Переключение между моделями**:
   - Можно настроить несколько провайдеров (Claude + Gemini)
   - Переключайтесь через настройки Cline

3. **Безопасность**:
   - Не делитесь API ключом
   - Храните его в безопасном месте
   - При необходимости можно создать новый ключ
